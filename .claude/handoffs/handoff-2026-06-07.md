## What shipped this session

### Bug fixes
- **Email-tab number input** — `|| 24` fallback caused field to snap on clear; `onFocus` auto-selects so typing replaces the value instead of appending. (`email-compose-form.tsx`)
- **Stale reminder display after save** — `saveReminderEdit` now calls `onEventUpdated(patchedEvent)`; `EmailActionsTab` keeps a `localEvents` copy and patches it immediately, so "Xh vorher" updates without a page reload.
- **Fragment key warning** — `sortedEvents.map` now uses `<Fragment key={event.id}>` on the outer element.
- **Edge runtime crash** — `signed-token.ts` used Node.js `crypto` (`createHmac`, `timingSafeEqual`), which is unavailable in the Next.js Edge runtime that `middleware.ts` runs in. Rewrote using `crypto.subtle` (Web Crypto API). `sign`/`verify` are now async; cascaded `await` through `admin-session`, `admin-token`, `admin-api-gate`, middleware, and four route handlers. Algorithm (HMAC-SHA256), token format, and timing-safety are unchanged.
