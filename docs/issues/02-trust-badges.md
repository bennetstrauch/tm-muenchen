## Parent

Conversion optimisation — landing page overhaul (see docs/prd/conversion-optimisation.md)

## What to build

A new `TrustBadges` component — a horizontal strip of six quick-scan badges rendered directly between the Hero and the ForWhom section on all theme pages.

Badges (in order): Persönlich unterrichtet · Einfach erlernbar · Keine Konzentration nötig · Ohne Gedanken stoppen · Von Millionen weltweit praktiziert · 400+ wissenschaftliche Studien

On mobile the strip scrolls horizontally; on desktop the badges are centred in a single row. The component is stateless with no props.

## Acceptance criteria

- [ ] TrustBadges component exists and renders all six badges
- [ ] Strip appears between Hero and ForWhom on the main page (`/`) and all theme pages
- [ ] On mobile (390px) the badges are horizontally scrollable without clipping
- [ ] On desktop the badges are centred and fit in one row
- [ ] Site builds without errors
